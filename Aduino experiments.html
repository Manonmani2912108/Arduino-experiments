How to connect and use your Arduino

  1. Connect the Arduino Board
    Use a USB cable to connect the Arduino board to your computer or laptop.

  2. Open Arduino IDE
    Download and open the Arduino IDE software on your computer.

  3. Select the Board
    Go to Tools → Board and select your Arduino board (Example: Arduino Uno).

  4. Select the Port
    Go to Tools → Port and choose the correct COM port.

  5. Write or Paste the Code
    Type your Arduino code or paste an experiment program into the IDE.

  6. Upload the Code
    Click the Upload button to send the code to the Arduino board.

  7. Observe the Output
    Watch the LEDs, sensors, or connected components perform the experiment.
